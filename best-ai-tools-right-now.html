The Best AI Tools Right Now, and What Each One Is Actually Good At

11 min read · devnotes

There are now hundreds of AI tools competing for your attention, and most "best AI tools" lists just dump names on you without saying which one actually fits your task. Here's the practical version: what each major tool is genuinely best at, so you stop paying for — or fumbling with — the wrong one.

Start with the question, not the tool

No single AI does everything best. The honest starting point is picking based on what you're actually trying to accomplish this week, not which tool has the most hype. Most people who use AI daily end up running two tools, not one — a general assistant for writing and thinking, plus one specialist for whatever they do most (code, images, video, or research).

General writing, thinking, and everyday assistance

ToolBest for
ChatGPTThe broadest all-rounder — writing, analysis, and it also handles image and some video generation in one place. Good default if you only want one subscription.
ClaudeLong-form writing quality and very large documents — contracts, books, long reports — plus strong coding ability in the same conversation.
GeminiDeep research tasks and tight integration if you already live inside Google Docs, Sheets, and Gmail.

If you write or code daily, the general consensus among people who've tested extensively is that the writing and reasoning quality edge goes to Claude; if you want the single tool with the broadest range of built-in features, ChatGPT remains the default choice for most people.

Coding

Coding tools have split into a few distinct styles rather than one winner:

  • IDE-integrated tools (GitHub Copilot, Cursor) — live inside your code editor and understand your whole project, not just the one file you're looking at. Cursor in particular has become the go-to AI-native editor for developers who work in code daily.
  • Terminal-based agents (Claude Code) — best for complex, multi-file changes across a real codebase, run from the command line.
  • Chat-based coding (asking ChatGPT or Claude directly in a chat window) — more flexible for quick snippets and learning, but doesn't know your actual project files the way IDE tools do.
quick tip
If you're just starting to code (like in our HTML & CSS or JavaScript courses), a chat-based assistant is genuinely the better starting point — IDE-integrated tools shine once you have a real, multi-file project to manage.

Image generation

ToolBest for
MidjourneyThe strongest artistic and aesthetic quality — concept art, mood boards, stylized visuals. Paid only, no free tier.
DALL-E (via ChatGPT)Fast iteration and the most accessible option if you already use ChatGPT — also handles text inside images better than most.
Stable Diffusion / FluxFree and open-source, with the most technical control — best if you're comfortable with a steeper learning curve.

Video generation

This is the category that's changed the most recently. Veo and Sora currently lead for generating short, cinematic clips directly from a text description — both are strong for storytelling-driven content rather than simple talking-head videos, which are better served by avatar-specific tools like Synthesia.

Research and working with your own documents

  • Perplexity — best when you need citations and sources alongside the answer, useful for fact-checking.
  • NotebookLM — best when you want an AI that only answers from documents you upload yourself, rather than pulling from the open internet.

Voice and audio

ElevenLabs is the current standout for realistic AI voice generation, text-to-speech, and voice agents — useful if you're making videos (see our guide on using AI for video) and don't want to record narration yourself.

The realistic way to build your own "AI stack"

  1. Ask yourself which task actually eats the most of your time each week — writing, coding, images, or video.
  2. Pick one general-purpose tool (ChatGPT, Claude, or Gemini) as your daily driver.
  3. Add exactly one specialist tool for your most time-consuming task, not five "just in case."
  4. Reassess every few months — this category moves fast, and this year's best pick won't necessarily be next year's.
bottom line
There's no single "best AI" — there's a best tool for the specific thing you're doing right now. Match the tool to the task, start with one general assistant plus one specialist, and don't let subscription hype talk you into paying for four tools that overlap.
